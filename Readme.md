# Step to deploy

## steps needed for installation

> sh install.sh

## how to configure the database

1 - copy the .env.example file to .env
2 - change the url MONGODB_URI, to the localhost of the database instance server
3 - run mongod on shell or cmd

## build application

> yarn start
